# Projects

Dossier des **projets réels** hébergés dans la SFIA.

Chaque projet dispose de son propre sous-dossier et, le cas échéant, de son dépôt Git dédié.

## Règles

- Un projet = un objectif clair, un cycle de vie traçable dans Notion
- Lier le projet à la base Notion **Projects**
- Ne pas mélanger le code de plusieurs projets dans un même dépôt
