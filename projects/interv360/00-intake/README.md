# 00 — Intake

Phase d'**intake** : collecte initiale du besoin, du contexte et des parties prenantes.

## Rôle

- Identifier le problème ou l'opportunité
- Recenser les acteurs et leurs attentes
- Documenter les contraintes connues à ce stade
- Préparer le passage au cadrage métier

## Statut

Phase active — en cours.
