# SFIA — Controlled Delivery

Controlled Delivery est un **standard SFIA de delivery contrôlé**.

Il permet d'adapter le niveau de contrôle au niveau de maturité d'un sujet :

- **phase 1 — cadrage fin** : micro-cycles lorsque le périmètre est instable ;
- **phase 2 — réalisation groupée contrôlée** : lot cohérent lorsque les règles sont stabilisées.
