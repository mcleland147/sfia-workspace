# SFIA Fast Track

## 1. Objectif

SFIA Fast Track est la méthode de livraison rapide, contrôlée et capitalisable utilisée pour créer des applications ou produits numériques avec assistance IA.

Elle vise à combiner :

- cadrage clair ;
- cycles courts ;
- garde-fous explicites ;
- PR ciblées ;
- validation systématique ;
- documentation continue ;
- automatisation progressive.

## 3. Règles clés

- un cycle = un objectif ;
- PR unique par cycle ;
- post-merge documenté ;
- audit avant cycle majeur ;
- REX après jalon majeur ;
- Figma Design-first pour UI premium ;
- Notion uniquement après plan validé ;
- backend/API dans cycle dédié ;
- automatisation progressive.

## 4. Workflow standard

1. Cadrer le besoin ;
2. Ouvrir une branche dédiée ;
3. Définir livrable et garde-fous ;
4. Exécuter le cycle ;
5. Valider diff/build/tests ;
6. Commit ;
7. PR ;
8. Merge ;
9. Post-merge ;
10. REX ou capitalisation si nécessaire.
