# SFIA Fast Track — Notion Publication Checklist

## 1. Préparation

- [ ] Audit documentaire réalisé
- [ ] Arborescence cible validée
- [ ] Documents publiables identifiés
- [ ] Documents internes exclus
- [ ] Documents à adapter listés

## 2. Qualité éditoriale

- [ ] Titres adaptés
- [ ] Statuts Git/merge retirés ou résumés
- [ ] Contenu sensible exclu
- [ ] Structure lisible
- [ ] Liens vérifiés

## 3. Validation

- [ ] Go humain obtenu
- [ ] Dry-run réalisé si disponible
- [ ] Sync exécutée uniquement après validation
- [ ] Rendu Notion vérifié

## 4. Après publication

- [ ] Rapport publication produit
- [ ] Écarts documentés
- [ ] Corrections prévues si nécessaire
