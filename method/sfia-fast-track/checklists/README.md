# SFIA Fast Track — Checklists

This folder contains operational checklists used to apply SFIA Fast Track rules, guardrails and validation criteria.

## Checklists

| Document | Role |
|----------|------|
| `sfia-validation-checklist.md` | Operational checklist for validating SFIA prompts, repository execution, diffs, commits, deliverables, PR readiness, post-merge, capitalization and Notion preparation |
| `validation-checklist.md` | General cycle validation checklist |
| `guardrails-checklist.md` | Guardrails verification checklist |
| `notion-publication-checklist.md` | Notion publication preparation checklist |

## v1.1 Checklist Status

`sfia-validation-checklist.md` is the canonical SFIA v1.1 operational validation checklist.

The following checklists are preserved as legacy or specialized references:

- `validation-checklist.md`
- `guardrails-checklist.md`
- `notion-publication-checklist.md`

They should not override `sfia-validation-checklist.md` for v1.1 cycle validation.
