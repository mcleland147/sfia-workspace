# SFIA Fast Track — Validation Checklist

## 1. Git

- [ ] Branche correcte
- [ ] `git status --short` vérifié
- [ ] `git diff --stat` vérifié
- [ ] Aucun fichier hors scope modifié

## 2. Code

- [ ] Build exécuté si code modifié
- [ ] Tests exécutés si code modifié
- [ ] Aucun backend/API modifié si hors scope
- [ ] Aucun script modifié si hors scope
- [ ] Aucune CI modifiée si hors scope

## 3. Documentation

- [ ] Livrable documentaire créé ou mis à jour
- [ ] Roadmap mise à jour si nécessaire
- [ ] Statut clair
- [ ] Prochaine étape documentée

## 4. PR

- [ ] Body PR préparé
- [ ] Garde-fous listés
- [ ] Validations listées
- [ ] Hors scope listé

## 5. Post-merge

- [ ] Commit d'intégration identifié
- [ ] Statut de merge documenté
- [ ] Roadmap mise à jour
- [ ] `main` poussé
