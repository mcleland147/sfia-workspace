# SFIA Platform — Methods

Ce dossier contient les méthodes plateforme SFIA par rôle, discipline ou pratique.

Il ne correspond pas à la méthode active de delivery.

## Distinction

| Zone | Rôle |
|------|------|
| `method/sfia-fast-track/` | Méthode active de delivery SFIA |
| `methods/` | Méthodes complémentaires |
| `docs/methods/` | Méthodes plateforme, rôles, disciplines, pratiques |
