# SFIA Role Methods

Ce dossier contient les **méthodes SFIA par rôle**.

Chaque fichier décrit :

- la mission du rôle ;
- les moments où le mobiliser ;
- la valeur projet attendue ;
- la valeur SFIA attendue ;
- les questions clés ;
- les livrables typiques ;
- le savoir-faire ;
- la posture SFIA / savoir-être ;
- les erreurs à éviter ;
- l'évolution continue.

Les méthodes par rôle sont des **templates vivants**, enrichis par les projets pilotes.
